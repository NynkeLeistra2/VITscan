# VIT-scan

Webapp voor de VIT-scan van Nynke Leistra Coaching en Advies. Zie `CLAUDE.md` voor projectcontext en `VIT-scan-projectplan.md` voor het volledige plan.

## Starten

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

Next.js (App Router) + TypeScript + Tailwind + Supabase (EU-regio).
